## Lingkup Pekerjaan

Tim teknisi kami akan menangani seluruh proses instalasi fisik dan digital di lokasi Anda, meliputi:

1.  **Pemasangan Hardware:** Perakitan Kiosk, pemasangan Bracket TV, dan penarikan kabel LAN/HDMI.
2.  **Konfigurasi Jaringan:** Setup IP Address statis, konfigurasi Mikrotik (jika ada), dan tes koneksi server lokal.
3.  **Instalasi Software:** Deploy aplikasi antrian di server lokal terminal atau Cloud.
4.  **Integrasi System:** Koneksi API antara mesin antrian dengan sistem booking tiket dan manifest penumpang.

Kami menjamin instalasi yang rapi, aman (electrical safety), dan sesuai standar estetika terminal transportasi.
